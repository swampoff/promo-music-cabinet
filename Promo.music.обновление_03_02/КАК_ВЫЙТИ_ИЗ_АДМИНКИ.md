# 🚪 КАК ВЫЙТИ ИЗ АДМИНКИ

## ⚡ БЫСТРЫЙ ОТВЕТ

### Способ 1: Кнопка "Выйти" в боковом меню (ОСНОВНОЙ)

**Desktop версия:**
```
1. Посмотрите в ЛЕВОЕ боковое меню
2. Прокрутите вниз (если нужно)
3. Внизу меню найдёте кнопку:
   
   ┌────────────────────────┐
   │  🚪 Выйти              │
   └────────────────────────┘
   
4. Нажмите на неё
5. Готово! Вы на странице логина
```

**Mobile версия:**
```
1. Нажмите кнопку меню ☰ (гамбургер) в верхнем левом углу
2. Откроется боковое меню
3. Прокрутите вниз
4. Внизу найдёте кнопку "🚪 Выйти"
5. Нажмите на неё
6. Готово! Вы на странице логина
```

---

### Способ 2: Через URL (если кнопка не видна)

Просто перейдите на главную страницу:
```
Измените URL в браузере с:
https://.../#/admin

На:
https://.../

И нажмите Enter
```

---

### Способ 3: Через консоль разработчика (для разработчиков)

```javascript
// Откройте консоль (F12) и выполните:
localStorage.removeItem('isAuthenticated');
localStorage.removeItem('userRole');
window.location.href = '/';
```

## 🐛 ЕСЛИ КНОПКА НЕ РАБОТАЕТ

### Проблема 1: Кнопка не видна
**Решение:** Прокрутите боковое меню вниз до конца

### Проблема 2: Кнопка не реагирует
**Решение:** 
1. Перезагрузите страницу (F5)
2. Попробуйте снова
3. Если не помогло - используйте Способ 2 (через URL)

### Проблема 3: После выхода снова в админке
**Решение:**
1. Очистите кеш браузера (Ctrl+Shift+Delete)
2. Закройте все вкладки с приложением
3. Откройте приложение в новой вкладке
4. Вы должны быть на странице логина

## 💡 СОВЕТЫ

### ✅ Хорошие практики:
- Всегда выходите через кнопку "Выйти"
- Не закрывайте просто вкладку (останетесь залогинены)
- Выходите перед закрытием браузера для безопасности

### ⚠️ Важно знать:
- После выхода нужно снова вводить credentials
- Все несохраненные данные будут потеряны
- Выход не влияет на данные в базе

### 🔐 Безопасность:
- Всегда выходите из админки на общих компьютерах
- Не оставляйте админку открытой без присмотра
- Используйте выход вместо просто закрытия вкладки

## ❓ ЧАСТЫЕ ВОПРОСЫ

### Q: Где находится кнопка выхода?
**A:** В самом низу левого бокового меню, красная кнопка с иконкой 🚪

### Q: Сохранятся ли мои данные после выхода?
**A:** Да! Все изменения в базе сохраняются. Выход только завершает сессию.

### Q: Можно ли выйти по горячим клавишам?
**A:** Нет встроенных горячих клавиш, используйте кнопку в меню.

### Q: Что делать если кнопка не работает?
**A:** Перейдите на главную страницу через URL или используйте консоль.

### Q: Нужно ли выходить каждый раз?
**A:** Рекомендуется выходить для безопасности, особенно на общих компьютерах.
