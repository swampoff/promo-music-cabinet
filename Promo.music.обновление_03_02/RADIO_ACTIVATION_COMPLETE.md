## 🚀 КАК АКТИВИРОВАТЬ РАДИОСТАНЦИЮ:

### **Способ 1: Из кабинета артиста**

1. Войдите как артист:
   ```
   Email: artist@promo.fm
   Password: artist123
   ```

2. В **левом sidebar** найдите **Workspace Switcher** (самая верхняя карточка)

3. Кликните на карточку → откроется меню всех кабинетов

4. Найдите **"Радиостанция"** (4-я в списке, indigo/purple цвет)
   - Иконка: 📻 Radio
   - Описание: "Трансляции и плейлисты"
   - **БЕЗ ЗАМКА** 🔓

5. Кликните → страница перезагрузится

6. **ВЫ В КАБИНЕТЕ РАДИОСТАНЦИИ!** 🎙️

---

### **Способ 2: Прямой вход через UnifiedLogin**

## 🐛 ЕСЛИ НЕ РАБОТАЕТ:

### **Проблема 1: Радиостанция все еще заблокирована 🔒**

**Решение:**
```bash
# Очистите кеш браузера
Ctrl+Shift+R (или Cmd+Shift+R)

# Или
localStorage.clear();
location.reload();
```

### **Проблема 2: При клике ничего не происходит**

**Решение:**
```javascript
// В консоли DevTools проверьте:
localStorage.getItem('userRole');  // должно быть 'radio_station'

// Если нет, установите вручную:
localStorage.setItem('userRole', 'radio_station');
location.reload();
```

### **Проблема 3: Открывается кабинет артиста вместо радио**

**Решение:**
```javascript
// Проверьте роль:
console.log(localStorage.getItem('userRole'));

// Должно быть: 'radio_station'
// Если 'artist' или 'admin' - сбросьте:
localStorage.setItem('userRole', 'radio_station');
location.reload();
```
