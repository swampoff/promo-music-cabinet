## 🔍 ПОИСК И ФИЛЬТРАЦИЯ

### Поиск:
- Введите текст в строку поиска
- Поиск ищет по:
  - Имени артиста
  - Названию трека
  - Жанру
- Работает в реальном времени

### Фильтры (5 кнопок):
```
[Все] [Новое] [В работе] [Разослано] [Архив]
```
- Кликните на кнопку для фильтрации
- Число в скобках = количество материалов

## 🔄 СОРТИРОВКА

### Как сортировать:

**Desktop:**
1. Кликните на заголовок столбца:
   - **Материал** - по имени артиста
   - **Дата** - по дате добавления
   - **Рассылок** - по количеству рассылок
2. Повторный клик меняет направление (↑↓)

**Mobile/Tablet:**
1. Выберите поле в dropdown:
   ```
   [По дате ▼] [↓]
   ```
2. Кликните стрелку для изменения направления

### Поля сортировки:
- **По дате** - новые первыми (default)
- **По артисту** - A→Z
- **По названию** - A→Z
- **По рассылкам** - больше первыми

## ☑️ МНОЖЕСТВЕННЫЙ ВЫБОР

### Как выбрать материалы:

**Desktop:**
1. Чекбокс в header = выбрать все
2. Чекбокс в строке = выбрать один материал

**Mobile:**
1. Чекбокс слева в каждой карточке
2. Выбранные карточки подсвечиваются голубым

### Визуальная индикация:
```
✅ Выбран   → голубая рамка, подсветка
☐ Не выбран → обычная рамка
```

## 🔧 МАССОВЫЕ ОПЕРАЦИИ

### Когда появляется панель:
```
Выбрано хотя бы 1 материал
  ↓
Появляется голубая панель сверху
```

### Доступные действия:

#### 1. **Экспорт** 📥
- Скачивает CSV файл с данными
- Содержит: ID, артист, название, тип, жанр, статус, дата, рассылки, файлы
- Открывается в Excel/Google Sheets

#### 2. **Архивировать** 📦
- Перемещает все выбранные в архив
- Статус меняется на "Архив"
- Показывает тост с количеством

#### 3. **Удалить** 🗑️
- Удаляет выбранные материалы
- Требует подтверждения
- **НЕЛЬЗЯ ОТМЕНИТЬ!**

#### 4. **Отмена** ✖️
- Снимает выбор со всех материалов
- Панель исчезает

## 📬 СОЗДАНИЕ РАССЫЛКИ

### Модал из 4 шагов:

#### Шаг 1: Выберите направление
```
┌──────────────┬──────────────┐
│ 📻 Радио     │ 🏢 Заведения │
├──────────────┼──────────────┤
│ 📰 СМИ       │ 💼 Лейблы    │
└──────────────┴──────────────┘
```
- Кликните на нужное направление
- Показывает количество баз

#### Шаг 2: Выберите базу рассылки
```
📻 Федеральные радиостанции     👥 45
Русское Радио, Европа Плюс...

🌐 Онлайн-радио                  👥 127
Интернет-радиостанции...
```
- Отображаются только базы выбранного направления
- Показывает количество контактов
- Клик выбирает базу

#### Шаг 3: Выберите файлы
```
✅ track.mp3              3.5 MB
✅ cover.jpg              1.2 MB
☐ press_release.pdf       0.8 MB
```
- Отметьте галочками нужные файлы
- Минимум 1 файл

#### Шаг 4: Комментарий (опционально)
```
┌────────────────────────────────────┐
│ Введите комментарий к рассылке...  │
│                                    │
│                                    │
└────────────────────────────────────┘
```
- Не обязательно
- Будет виден в отчёте

### Preview (автоматически):
```
ℹ️ Предпросмотр рассылки
━━━━━━━━━━━━━━━━━━━━━━━━━━
Направление:   Радиостанции
База:          Федеральные радиостанции

- **Отмена** - закрыть без отправки
- **Отправить рассылку** - создать рассылку

### После отправки:
```

• Статус: НОВОЕ → В РАБОТЕ (если было новое)

## 💾 ЭКСПОРТ ДАННЫХ

### Экспорт всех материалов:

**Desktop:**
1. Кнопка "Экспорт всех данных" (правый верхний угол)
2. Автоматически скачивается CSV
3. Имя файла: `pitching_full_export_[timestamp].csv`

### Экспорт выбранных:
1. Выберите материалы (чекбоксы)
2. Кликните "Экспорт" в Bulk Actions
3. Скачивается CSV только выбранных
4. Имя файла: `pitching_export.csv`

### Формат CSV:
```csv
ID,Артист,Название,Тип,Жанр,Статус,Дата добавления,Рассылок,Файлов
8001,The Hatters,Я делаю шаг,track,Alternative Rock,new,01.02.2026,0,2
...
```

## 📦 АРХИВИРОВАНИЕ

### Зачем нужен архив:
- Материалы, которые больше не актуальны
- Завершённые кампании
- Очистка основного списка

### Как архивировать:

**Single item:**
1. Кликните ⋮ (dropdown меню)
2. Выберите "В архив"
3. Материал перемещается в архив

**Bulk:**
1. Выберите материалы (чекбоксы)
2. Кликните "Архивировать" в панели
3. Все перемещаются в архив

### Как посмотреть архив:
1. Кликните фильтр **[Архив]**
2. Увидите все заархивированные материалы

### Как восстановить:
1. Откройте архив (фильтр)
2. Кликните "Восстановить" на материале
3. Материал возвращается в работу

## ❓ FAQ

### Q: Что значит "В работе"?
**A:** Материал, по которому была хотя бы 1 рассылка, но кампания ещё не завершена.

### Q: Можно ли отменить удаление?
**A:** Нет! Удаление необратимо. Используйте "Архив" вместо удаления.

### Q: Как узнать, кто получил рассылку?
**A:** В отчёте видно название базы и количество контактов. Детальный список контактов пока недоступен.

### Q: Зачем несколько рассылок одного трека?
**A:** Можно отправить на разные направления:
- Радио (45 контактов)
- СМИ (164 контакта)
- Заведения (245 контактов)
- И т.д.

### Q: Что такое Open Rate и Click Rate?
**A:**
- **Open Rate** - % получателей, открывших письмо
- **Click Rate** - % получателей, кликнувших по ссылкам

### Q: Где хранятся файлы?
**A:** Файлы привязаны к материалу. При удалении материала файлы тоже удаляются.

### Q: Можно ли редактировать отправленную рассылку?
**A:** Нет. Но можно отправить новую с обновлённой информацией.
