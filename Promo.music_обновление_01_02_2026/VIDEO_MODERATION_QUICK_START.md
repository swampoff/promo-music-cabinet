## 🚀 КАК ИСПОЛЬЗОВАТЬ

### 1. Просмотр видео на модерации:
```
Админ-панель → Модерация видео → "На модерации"
```

### 2. Одобрение видео:
```
Нажать "Одобрить" → Видео публикуется
→ Списываются средства (если платное)
→ Артист получает уведомление
```

### 3. Отклонение видео:
```
Нажать "Отклонить" → Ввести причину
→ Видео отклоняется
→ Артист получает уведомление с причиной
```

### 4. Массовые действия:
```
Выбрать чекбоксы → "Одобрить все" / "Отклонить все"
```

## 💡 ПОДСКАЗКИ

### Для тестирования:
1. Сейчас показываются демо-видео
2. Чтобы протестировать реальную модерацию:
   - Перейдите в Кабинет артиста
   - Загрузите видео
   - Вернитесь в админ-панель
   - Модерируйте загруженное видео

### Быстрые причины отклонения:
- Низкое качество видео
- Проблемы со звуком
- Нарушение авторских прав
- Несоответствие категории
- Неприемлемый контент
