## 📍 МЕСТО 1: DESKTOP - Верхний header (СПРАВА ВВЕРХУ)

**Самое заметное место!**

```
┌────────────────────────────────────────────────────────────┐
│  Дашборд                             [🚪 Выйти]           │
│  Admin CRM • Promo.music                   ↑               │
│                                         ЗДЕСЬ!             │
└────────────────────────────────────────────────────────────┘
```

**Как найти:**
1. Откройте админ-панель на компьютере (Desktop)
2. Посмотрите в **ПРАВЫЙ ВЕРХНИЙ УГОЛ**
3. Там большая красная кнопка с рамкой: **[🚪 Выйти]**
4. Нажмите на неё

**Стиль:**
- Красный текст
- Красная рамка
- Иконка выхода 🚪
- При наведении светится красным

│ Admin CRM        [🚪] [☰]     │
│                    ↑            │

**Как найти:**
